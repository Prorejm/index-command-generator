# 🧾 食指指令生成器 — 都市之令

> **「谨遵指令之意」** — 指令无需知晓来源，只需践行其意志。

基于《月亮计划》（Project Moon）世界观中 **食指（Index）** 帮派风格的随机指令生成器。每次刷新生成一道独一无二的都市指令，信纸样式、指令语气、都市细节皆忠实还原原作氛围。

## ✨ 功能特性

### 🎴 无限随机指令
通过 **7 大部件库** 的随机组合，可生成 **逾万种** 不同的都市指令：

| 部件 | 数量 | 说明 |
|------|:----:|------|
| 开场词 | 7 种 | 指令开篇的宣誓句式 |
| 动作 | 15 种 | 必须执行的行为（前往、交付、寻找、摧毁……） |
| 目标物 | 20 种 | 指令涉及的道具或信物 |
| 人物 | 14 种 | 都市中的各色收尾人与后巷居民 |
| 地点 | 15 处 | 巢、后巷、协会据点等真实都市场景 |
| 时限 | 10 种 | 深宵、日出、钟响等都市特色时间 |
| 条件 | 10 种 | 执行指令时必须遵守的规则 |
| 后果 | 8 种 | 违背后将面临的惩罚 |
| 落款 | 5 种 | 食指的结印语 |

> **组合数**：7×15×20×14×15×10×10×8×5 = **17.64 亿**+ 种可能，绝对不止"上千种"。

### 🌆 沉浸式都市视觉
- 背景采用都市夜景图，叠加暗色滤镜与颗粒飘浮效果
- 指令以 **泛黄信纸** 样式呈现，模拟食指白色信纸的质感
- 顶部带有 **食指朱红圆印**，底部加盖 **食指方印**
- 切换指令时带有淡入淡出的翻页动画

### 📋 实用功能
- **一键生成**：点击按钮或按 **空格键** 降下新指令
- **复录指令**：一键复制当前指令文本，方便分享或记录
- **计数器**：记录本次会话中已生成的指令数量

### 🌐 键盘快捷键
| 按键 | 功能 |
|------|------|
| `Space` / `空格键` | 生成新指令 |

## 🏙️ 世界观背景

**食指（Index / ???）** 是《月亮计划》宇宙中都市 **五指（Five Fingers）** 之一，以白长袍、佩剑与 **信纸指令** 为特征。

- 信纸指令会不知来源地出现在食指领地中
- 所有领地上的居民都必须完成指令内容
- 未及时完成指令者将被处决
- 食指内部有苦行者、代行者、传令员、纺织者等层级

本生成器所有地点、人物、物品均取材自《脑叶公司》《废墟图书馆》《边狱公司》等作品中描绘的都市设定，包括**巢**、**后巷**、**协会**、**翼公司**等标志性元素。

## 🚀 部署指南

### GitHub Pages 部署

1. **Fork 或克隆本仓库**
   ```bash
   git clone https://github.com/你的用户名/index-command-generator.git
   cd index-command-generator
   ```

2. **确保文件结构**
   ```
   index-command-generator/
   ├── index.html
   ├── city-bg.png
   └── README.md
   ```

3. **启用 GitHub Pages**
   - 进入仓库 **Settings → Pages**
   - Source 选择 **Deploy from a branch**
   - Branch 选择 **main**，目录选 **/(root)**
   - 点击 **Save**

4. **访问地址**
   部署完成后即可通过 `https://你的用户名.github.io/index-command-generator/` 访问。

### 本地运行
直接用浏览器打开 `index.html` 即可，无需任何服务器环境。

## 🎨 自定义指南

### 添加更多指令部件
打开 `index.html`，找到 `INGREDIENTS` 部分，在对应数组中追加内容即可：

```javascript
const LOCATIONS = [
  // ... 现有地点
  'G公司重力球穹顶实验室',  // 新增
  'J公司完全封锁区外围',    // 新增
];
```

### 更换背景图
替换 `city-bg.png` 文件即可。建议使用暗色调的都市/赛博朋克风格图片以获得最佳视觉效果。
