# 🧾 食指指令生成器 — 都市之令

> **「谨遵指令之意」** — 指令无需知晓来源，只需践行其意志。

基于《月亮计划》（Project Moon）世界观中 **食指（Index）** 帮派风格的随机指令生成器。每次刷新生成一道独一无二的都市指令，信纸样式、指令语气、都市细节皆忠实还原原作氛围。

在线体验：**https://prorejm.github.io/index-command-generator/**

---

## ✨ 功能特性

### 🎴 六种指令风格

| 风格 | 说明 | 灵感来源 |
|------|------|----------|
| **标准令** | 完整的都市指令格式：动作 + 目标 + 地点 + 人物 + 时限 + 条件 + 后果 | 月亮计划世界观标准指令 |
| **短令** | 极简箴言式："指令是投注"——附带一句意味深长的注解 | prescript 风格 / 食指暗语 |
| **列表令** | 3-6 条并列的具体行动指令，须同时执行 | 多重要务式指令 |
| **三行令** | 三段递进式指令，附有"行·知·成"等阶段标记 | 仪式感指令 |
| **谶言令** | 预言/警示风格的都市谶言，神秘而不可违逆 | 都市传说级警示 |
| **混令** | 随机混合以上五种风格，每次生成都不同 | 惊喜模式 |

### 🎴 部件库（标准令专用）

通过 **9 大部件库** 的随机组合，可生成 **逾17亿种** 不同的标准指令：

| 部件 | 数量 | 说明 |
|------|:----:|------|
| 开场词 | 9 种 | 指令开篇的宣誓句式 |
| 动作 | 20 种 | 必须执行的行为（前往、交付、寻找、摧毁、献上、掩埋……） |
| 目标物 | 28 种 | 指令涉及的神秘道具或信物 |
| 人物 | 18 种 | 都市中的各色收尾人与后巷居民 |
| 地点 | 18 处 | 巢、后巷、协会据点等真实都市场景 |
| 时限 | 14 种 | 深宵、日出、钟响、烟霾等都市特色时间 |
| 条件 | 13 种 | 执行指令时必须遵守的仪式化规则 |
| 后果 | 10 种 | 违背后将面临的惩罚 |
| 落款 | 6 种 | 食指的结印警示语 |

### 🖤 短令（prescript 风格）

借鉴 [nyos.dev/prescript](https://nyos.dev/prescript) 的设计语言，生成简短而深邃的指令暗语：

> **指令是投注**
> *投下便不可收回*

> **指令是沉默**
> *沉默本身就是回答*

> **指令是门槛**
> *跨过之后便不能回头*

### 🌆 沉浸式都市视觉
- 背景采用都市夜景图，叠加暗色滤镜与颗粒飘浮效果
- 指令以 **泛黄信纸** 样式呈现，模拟食指白色信纸的质感
- 顶部带有 **食指朱红圆印**「食」，底部加盖 **食指方印**
- 右上角显示当前指令风格标签
- 切换指令时带有淡入淡出的翻页动画

### 📋 实用功能
- **六种风格一键切换**：点击顶部标签栏切换不同指令类型
- **一键生成**：点击「降下新指令」按钮或按 **空格键** 刷新
- **复录指令**：一键复制当前指令文本，方便分享或记录
- **计数器**：记录本次会话中已生成的指令数量

### 🌐 键盘快捷键
| 按键 | 功能 |
|------|------|
| `Space` / `空格键` | 生成新指令（当前风格） |

## 🏙️ 世界观背景

**食指（Index / ???）** 是《月亮计划》宇宙中都市 **五指（Five Fingers）** 之一，以白长袍、佩剑与 **信纸指令** 为特征。

- 信纸指令会不知来源地出现在食指领地中
- 所有领地上的居民都必须完成指令内容
- 未及时完成指令者将被处决
- 食指内部有苦行者、代行者、传令员、纺织者等层级

本生成器所有地点、人物、物品均取材自《脑叶公司》《废墟图书馆》《边狱公司》等作品中描绘的都市设定，包括**巢**、**后巷**、**协会**、**翼公司**等标志性元素。

## 🚀 部署指南

### GitHub Pages（已部署）

项目已部署至：**https://prorejm.github.io/index-command-generator/**

如需部署到自己的仓库：

1. **Fork 或克隆本仓库**
   ```bash
   git clone https://github.com/你的用户名/index-command-generator.git
   cd index-command-generator
   ```

2. **确保文件结构**
   ```
   index-command-generator/
   ├── index.html
   ├── city-bg.png
   └── README.md
   ```

3. **启用 GitHub Pages**
   - 进入仓库 **Settings → Pages**
   - Source 选择 **Deploy from a branch**
   - Branch 选择 **main**，目录选 **/(root)**
   - 点击 **Save**

### 本地运行
直接用浏览器打开 `index.html` 即可，无需任何服务器环境。

## 🎨 自定义指南

### 添加更多指令部件
打开 `index.html`，找到 `INGREDIENTS` 部分，在对应数组中追加内容即可：

```javascript
const LOCATIONS = [
  // ... 现有地点
  'G公司重力球穹顶实验室',  // 新增
  'J公司完全封锁区外围',    // 新增
];
```

### 添加新的短令
在 `SHORT_MAIN` 和 `SHORT_DETAIL` 数组中追加配对条目：

```javascript
const SHORT_MAIN = [
  // ... 现有条目
  '指令是×××',  // 新增主标题
];
const SHORT_DETAIL = [
  // ... 现有条目
  '对应的注解文字',  // 新增注解
];
```

### 更换背景图
替换 `city-bg.png` 文件即可。建议使用暗色调的都市/赛博朋克风格图片以获得最佳视觉效果。
